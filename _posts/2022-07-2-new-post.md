## Building Project Base

### Motivation
Project Base is an idea I have had for a quite a while. This idea was born out of my frustration while learning programming. There should be a platform for learning
programming throught projects. Learning through projects allows a person to learn and then implement there skills quickly. While working through my CS degree I have
realized that I can only program while I am in a classroom setting. I am terrible at programming when it comes to an independent project. I don't even know where to 
start. So the idea with Project Base is to create a platform where people can learn programming throught real world problems rather than working through small challenge 
problems. Build real things for the real world.

### Vision
As of now the way I see Project Base working is that once a person signs up first you will need to determine their proficency in the subject. Once we do that we will 
have a list of projects based on their skill level. The goal of the project is not only for a person to create them on their own but also make a person think of the choices
they make during the project. For example if you are writing some function we will ask you "What kind of loop could you use here?" or "Could you use recurision here?".
The goal is to make a person independently think not hold their hand throught the project, otherwise it will just be the same as a classroom setting.
